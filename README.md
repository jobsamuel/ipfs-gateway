# IPFS-Gateway

> A simple IPFS gataway api.

## requirements

Before start, you need to initialize your firebase project by adding a json file called `.firebaserc` containing this:

```json
{
  "projects": {
    "default": "<YOUR-FIREBASE-PROJECT-ID>"
  }
}
```

## usage

```sh
# initialization
$ npm install

# local test
$ firebase serve --only functions
```
